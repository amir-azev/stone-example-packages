# Stone Example Packages

Example packages for the [Stone programming language](https://github.com/amir-azev/riva).

## Packages

### greeter

Simple greeting generator.

```stone
import { hello, welcome } from "greeter"

print(hello("World"))     // Hello, World!
print(welcome("Alice"))   // Welcome, Alice! Nice to meet you.
```

## Install

```bash
stone install greeter --source https://raw.githubusercontent.com/amir-azev/stone-example-packages/main
```
